# Feature: GitHub Actions CI

## Branch: `feat/ci`

## Context

Every regression caught so far was found by manual testing after merge. CI should
run on every PR and block merging if any check fails. The pipeline must cover
Rust, TypeScript, and the sidecar build — matching the same checks developers run
locally per AGENTS.md.

## Chunks

### Chunk 41 — Core CI Pipeline

- [ ] Create `.github/workflows/ci.yml`
- [ ] Trigger on: `push` to `main`, `pull_request` targeting `main`
- [ ] Single job: `check` running on `windows-latest` (matches developer environment)
- [ ] Steps in order:
  1. `actions/checkout@v4`
  2. `dtolnay/rust-toolchain@stable` — install Rust stable
  3. `actions/cache@v4` — cache `~/.cargo/registry`, `~/.cargo/git`,
     `src-tauri/target` keyed on `Cargo.lock`
  4. `oven-sh/setup-bun@v2` — install Bun
  5. `actions/cache@v4` — cache `node_modules` keyed on `bun.lock`
  6. `bun install` — install frontend dependencies
  7. `bun run build:sidecar` — compile sidecar binary (required before cargo build)
  8. `bun run typecheck` — TypeScript type check
  9. `bun run lint` — ESLint
  10. `cargo build --manifest-path src-tauri/Cargo.toml` — Rust compile check
  11. `cargo test --manifest-path src-tauri/Cargo.toml` — Rust tests
  12. `bun run test` — frontend Vitest tests

### Chunk 42 — PR Status Checks & Branch Protection

- [ ] Add `.github/pull_request_template.md` with sections:
  - `## Summary` — what this PR does
  - `## Checklist` — mirrors the AGENTS.md verification checklist (checkboxes)
  - `## Tested` — how it was manually tested
- [ ] Document in `AGENTS.md` under a new `## CI` section:
  - All PRs must pass CI before merge
  - Branch protection rules to enable on `main`: require status checks, require
    PR, no direct pushes
  - How to run CI checks locally: `bun run typecheck && bun run lint && cargo test
    --manifest-path src-tauri/Cargo.toml && bun run test`

## Verify

- Push a branch with a deliberate TypeScript error — CI must fail on `typecheck`
- Push a branch with a deliberate Rust compile error — CI must fail on `cargo build`
- Push a clean branch — all steps pass and PR is mergeable
- Cache hits on second run reduce total CI time by at least 50%
